# Shipitsin
Алгоритм построения фрактала Серпинского начинается с задания исходного равностороннего треугольника, ориентированного вершиной вверх: my_points = [[-200, -100], [0, 200], [200, -100]].
 Функция draw_triangle(points, color, my_turtle) перемещает черепаху к первой вершине без рисования, последовательно соединяет три точки, образуя замкнутый контур, и заливает его указанным цветом с помощью begin_fill() и end_fill(). Ключевым элементом рекурсивного деления является функция get_mid(p1, p2), вычисляющая середину отрезка между двумя точками по формуле [(p1[0] + p2[0]) / 2, (p1[1] + p2[1]) / 2]. Основная логика реализована в рекурсивной функции sierpinski(points, degree, my_turtle): если degree == 0, рисуется текущий треугольник; в противном случае вычисляются середины всех трёх сторон, формируя внутренний «пустой» треугольник, и рекурсивно вызываются три подтреугольника — по одному в каждом углу исходного. На каждом уровне глубины используется уникальный цвет из списка colormap, создавая радужный визуальный эффект. Графическое окно настроено с чёрным фоном (bgcolor("black")) для контраста, черепаха скрыта (hideturtle()), скорость отрисовки максимальна (speed(0)), а окно закрывается по клику (exitonclick()). Глубина рекурсии установлена в degree = 5, что обеспечивает высокую детализацию фрактала без избыточной нагрузки на систему, давая чёткий, симметричный и эстетически выразительный
